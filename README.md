# Week 6: Data Visualization Mastery with Seaborn

## Project Overview
This project creates an Interactive Sales Dashboard using Python, Seaborn, Matplotlib, and Plotly.

The dashboard provides insights into:
- Sales trends
- Product performance
- Regional sales analysis
- Data distribution
- Correlation analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
