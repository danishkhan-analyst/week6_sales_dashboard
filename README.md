# Week 6: Data Visualization Mastery with Seaborn

## Project Overview

This project creates an Interactive Sales Dashboard using Python, Seaborn, Matplotlib, and Plotly.

The dashboard provides insights into:

- Sales trends
- Product performance
- Regional sales analysis
- Data distribution
- Correlation analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Setup Instructions

### 1. Install Python

Download and install Python from the official website.

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run the Dashboard Script

```bash
python dashboard.py
```

### 4. Open the Interactive Dashboard

Open the generated file:

```bash
visualizations/dashboard.html
```

in your web browser.

## Visualizations Included

### 1. Sales Trend Analysis

A line chart showing how total sales change over time.

**Purpose:**
- Identify sales growth and decline
- Detect trends and patterns
- Monitor overall business performance

### 2. Product Sales Analysis

A bar chart comparing sales across products.

**Purpose:**
- Identify top-performing products
- Compare product performance
- Support product-related business decisions

### 3. Regional Sales Analysis

A pie chart showing sales contribution by region.

**Purpose:**
- Compare regional performance
- Identify high-performing regions
- Support regional business strategies

### 4. Sales Distribution (Boxplot)

A boxplot displaying the distribution of sales values.

**Purpose:**
- Detect outliers
- Understand data spread
- Analyze sales variability

### 5. Correlation Heatmap

A heatmap showing relationships between numerical variables.

**Purpose:**
- Identify correlations
- Discover hidden patterns
- Support data-driven decision making

### 6. Interactive Sales Dashboard

An interactive Plotly dashboard that allows users to:

- Hover over data points
- Zoom in and out
- Explore sales trends interactively
- Compare regional performance

## Dashboard Interpretation

### Sales Trends

The sales trend visualization helps track business performance over time and identify periods of growth or decline.

### Product Performance

The product sales chart highlights which products contribute the most revenue and helps identify best-selling products.

### Regional Analysis

The regional sales visualization shows how different regions contribute to total sales and helps compare market performance.

### Sales Distribution

The boxplot helps understand the spread of sales data and detect unusual observations or outliers.

### Correlation Analysis

The heatmap reveals relationships between variables and helps identify factors that may influence sales performance.
